# План проекта «Миры Двуликого: Солнечная система»

## Описание

Интерактивная модель вымышленной бинарной звёздной системы, выполняемая как **2D-визуализация с видом сверху** (как карта). В центре — два солнца, вращающихся друг вокруг друга. Вокруг них обращаются несколько планет. Одна из них — аналог Сатурна с кольцами и 7 населёнными спутниками.

## Технологии

- **HTML5** — структура страницы
- **CSS3** — стили интерфейса
- **JavaScript (ES6+)** — логика приложения
- **Three.js** — 2D-рендеринг через WebGL (ортографическая камера, вид сверху)
- **OrbitControls** — управление масштабированием и перемещением камеры
- **Webpack** — сборка проекта, бандлинг модулей, оптимизация ассетов
- **Docker Compose** — контейнеризация среды разработки и развёртывания

## Ключевые элементы системы

### Звёзды (2)
- **Жёлтое солнце** — крупное, яркое, цвет #FFD700 (золотисто-жёлтый), эффект свечения
- **Красное солнце** — поменьше, цвет #FF4500 (оранжево-красный), эффект свечения
- Вращаются вокруг общего центра масс по эллиптическим орбитам
- Эффект свечения (glow)

### Планеты (6)
1. **Внутренняя планета** — небольшая, горячая, близкая к солнцам
2. **Вторая планета** — средняя, скалистая
3. **Планета-Сатурн** — газовый гигант с кольцами (центральный объект)
4. **Четвёртая планета** — газовый гигант, поменьше
5. **Пятая планета** — ледяной гигант
6. **Внешняя планета** — далёкая, маленькая

### Планета-Сатурн (детали)
- Крупный газовый гигант с текстурой в тёплых тонах
- Система колец (внутреннее, среднее, внешнее) — вид как кольца
- Орбита на среднем расстоянии от солнц

### Спутники Сатурна (7 населённых)
1. **Мимас** — «Мир Императора». Мир с развитой цивилизацией, имперский город-государство.
2. **Энцелад** — планета разумного существа, огромный океан живой субстанции. Весь мир — один организм.
3. **Тефия** — полузатопленный мир, портовые города на островах и побережьях.
4. **Диона** — *TBD (будет определено позже)*
5. **Рея** — «Мир Песнопевцев и Слепцов». Зелёный мир с растительностью, джунгли, грибы, споры.
6. **Титан** — скалистый мир алмазов и суровых воинов.
7. **Япет** — «Мир Сумерек и Рассвета». Тёмный и ледяной мир с биолюминесценцией ночью; пылающая огнём пустыня днём.

## Интерактивность

### Камера
- **Вид всегда сверху** (ортографическая камера)
- Масштабирование: pinch/scroll
- Перемещение: drag
- Фиксированный ракурс — полноценное 3D не требуется

### Временная шкала
- Ползунок для перемещения по времени
- Кнопки: пауза / воспроизведение
- Регулировка скорости времени
- Отображение текущей временной метки
- В любой момент можно «застыть» и увидеть расположение всех тел

### Выбор точки на спутнике
- Выбор точки на поверхности спутника = **выбор точки на круге** (вид сверху)
- При клике по кругу спутника определяется угол на поверхности
- Отображение **конуса видимости** на небосводе:
  - Конус рисуется как сектор на круге небосвода вокруг спутника

### Навигация по масштабу
- **Общий вид:** вся бинарная система
- **Планета-Сатурн:** приближение к Сатурну и его орбите
- **Спутник:** приближение к выбранному спутнику для выбора точки

### Панель информации
- Название, тип, радиус, температура, население
- Описание

## Сборка и развёртывание

### Webpack

**Назначение:** модульная сборка проекта, бандлинг JS/CSS, обработка ассетов.

**Структура:**
- `webpack.config.js` — конфигурация сборки
- `src/` — исходный код (модули)
- `dist/` — результат сборки (готовые к деплою файлы)

**Конфигурация:**
- **entry** — `src/main.js` (точка входа)
- **output** — `dist/bundle.js`
- **modules:**
  - `babel-loader` — транспиляция JS (ES6+ → совместимый JS)
  - `css-loader` + `style-loader` — обработка CSS-модулей
  - `file-loader` / `asset/resource` — обработка изображений и шрифтов
- **plugins:**
  - `HtmlWebpackPlugin` — генерация `index.html` из шаблона
  - `MiniCssExtractPlugin` — вынос CSS в отдельный файл
  - `CleanWebpackPlugin` — очистка `dist/` перед каждой сборкой
- **devServer** — горячая перезагрузка (HMR) при разработке
- **mode** — `development` / `production` (с минификацией и tree-shaking)

**npm-скрипты:**
- `npm run build` — продакшн-сборка
- `npm run dev` — разработка с HMR (webpack-dev-server)
- `npm run lint` — проверка кода (ESLint)

### Docker Compose

**Контейнеры:**

1. **app (разработка)**
   - **Образ:** `node:20-alpine`
   - **Назначение:** среда разработки с webpack
   - **Объём:** `./src → /app/src` (горячая подгрузка исходников)
   - **Порты:** `8080:8080` (webpack-dev-server)
   - **Томары:** кэш npm в volume для скорости
   - **Команда:** `npm run dev`

2. **nginx (продакшн)**
   - **Образ:** `nginx:alpine`
   - **Назначение:** раздача статических файлов
   - **Объём:** `./dist → /usr/share/nginx/html` (собранные файлы)
   - **Порты:** `80:80` (HTTP), `443:443` (HTTPS, при наличии SSL)
   - **Конфиг:** `nginx.conf` — gzip-сжатие, кэширование, CORS-заголовки

3. **node_modules (опционально)**
   - **Volume:** `npm-cache` и `node_modules` в именованных volume
   - **Назначение:** избегание повторной установки зависимостей, изоляция от хоста

**docker-compose.yml:**
```yaml
version: '3.8'

services:
  app:
    build:
      context: .
      dockerfile: Dockerfile
    container_name: tfwcosmo-dev
    working_dir: /app
    volumes:
      - ./src:/app/src
      - /app/node_modules
    ports:
      - "8080:8080"
    command: npm run dev
    environment:
      - NODE_ENV=development

  nginx:
    image: nginx:alpine
    container_name: tfwcosmo-prod
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./dist:/usr/share/nginx/html:ro
      - ./nginx.conf:/etc/nginx/nginx.conf:ro
    depends_on:
      - app
    restart: unless-stopped
```

**Dockerfile (app):**
```dockerfile
FROM node:20-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
EXPOSE 8080
CMD ["npm", "run", "dev"]
```
